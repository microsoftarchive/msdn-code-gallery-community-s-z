画面のボタンを押すことで、画像フォルダ以下にあるjpgファイルの中からランダムに画面に画像を表示するサンプルプログラムです。

ポイントは以下のようなものがあります。

  • システムの特殊なフォルダーのパスの取得方法(Enviroment.GetFolderPath)
  • 指定したフォルダ以下のファイルの検索方法(Directory.GetFiles)
  • 画像へのファイルパスからBitmapImageの作成方法
