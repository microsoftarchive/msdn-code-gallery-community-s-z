This sample demonstrates how to make use of a static/dynamic C++ library to build an end-end Android Java Native Interface (JNI) application in Visual Studio 2015.

Description:

This sample is a port of the Android NDK TwoLibs sample. It is called this because it uses a C++ Android native library from a Java Native Interface app.

This sample is a good starting point for developers to understand how to use static/dynamic shared libraries  to build an end-to-end JNI Android application with Visual Studio 2015.

This sample is written in C/C++ and Java.

Build the sample

To build this sample, open the solution (TwoLibs.sln) file from Visual Studio . Press F7 or go to Build->Build Solution from the top menu after the sample has loaded. The application is supported for x86 and ARM configurations, and has been tested for API Level 19 specifically.

Run the sample

To run this sample after building it, press F5 (run with debugging enabled) or Ctrl+F5 (run without debugging enabled) from Visual Studio 2015. (Or select the corresponding options from the Debug menu.)
